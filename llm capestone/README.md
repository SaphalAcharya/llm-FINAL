# IMDB Movie Review Sentiment Analysis

This project performs **binary sentiment classification** on the **IMDB
50K Movie Reviews Dataset**.

## Dataset Information

Dataset: IMDB Dataset of 50K Movie Reviews\
Source:
https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

Contains 50,000 labelled movie reviews (positive/negative). Balanced
dataset.

## Techniques & Methods Used

### Data Preprocessing

-   HTML tag removal
-   Lowercasing
-   Punctuation removal
-   Stopword removal
-   Tokenization
-   Lemmatization
-   Train-test split

### Feature Engineering

-   Bag of Words (BoW)
-   TF-IDF
-   Word2Vec
-   GloVe
-   Keras Tokenizer + Embedding layer

### Models Implemented

**Machine Learning Models** - Logistic Regression - Naive Bayes - SVM -
Random Forest - XGBoost


### Evaluation Metrics

-   Accuracy
-   Precision
-   Recall
-   F1 Score
-   Confusion Matrix
